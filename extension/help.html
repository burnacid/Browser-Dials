Quick Start

  1. Open your browser extensions page and load the extension folder as unpacked.
  2. Open a new tab to see Browser Dials.
  3. Use the plus card in the grid to add your first dial.
  4. Open Settings to configure search, splash background, and sync mode.

Self-Hosted Sync Server

Use this if you want your dials synced across multiple browsers or devices.

Prerequisites

  • Docker 20+
  • Docker Compose 2+

Simple Docker setup

cp examples/docker-compose.simple.yml docker-compose.yml
cp examples/.env.simple .env
# Edit .env and set strong passwords + API_KEY
docker-compose up -d
docker-compose exec server node scripts/create-user.js admin your-password

Windows quick setup

.\examples\quickstart.ps1 -Mode simple
# Then edit .env and run:
docker-compose up -d

Verify server is running

docker-compose ps
curl http://localhost:3737/api/sync

The curl call should return Unauthorized when no auth headers are passed. That means the server is up.

Connect Extension to Sync

  1. Open Settings in Browser Dials.
  2. Set Mode to Sync to server.
  3. Server URL: http://localhost:3737 (or your deployed server URL).
  4. Paste the same API key you set in your server .env.
  5. Enter your sync username and password.
  6. Click Save Sync Settings, then Test Connection.
  7. Use Force Sync Now when you want to manually pull/push immediately.

Profiles

Profiles help you separate bookmarks by context (work, personal, travel, learning, etc.).

  • Create profile: use the plus button in the top bar.
  • Rename profile: use the pencil button in the top bar or from Settings.
  • Delete profile: open Settings and remove it from the Profiles list.
  • Switch profile: click the profile tab in the top bar.

Dials and Folders

Create a dial

  1. Click the Add dial card.
  2. Enter title and URL.
  3. Optionally upload a custom icon.
  4. Save.

Create a folder

  1. Click Add dial.
  2. Enable Create as folder.
  3. Set folder name and save.
  4. Open the folder and use Add link to add items inside.

Organize quickly

  • Drag and drop dials to reorder.
  • Drag a dial onto another dial to merge into a folder.
  • Use right-click on cards for edit/delete actions.

Search and Splash

  • Search bar can be shown or hidden in Settings.
  • Click the search logo to switch search engines from the dropdown.
  • Splash background supports uploaded images or public providers.
  • You can tune splash opacity and refresh public background manually.

Backup and Restore

  • Export JSON from Settings to back up your current data.
  • Import JSON to restore a previous backup.
  • Use this before big cleanup changes or migrations.

Troubleshooting

Sync test fails

  • Confirm server URL, API key, username, and password are correct.
  • Check server logs: docker-compose logs -f server.
  • Check service status: docker-compose ps.

Cannot reach server from browser

  • Verify the server port is exposed and not in use by another app.
  • Try opening http://localhost:3737/api/sync directly in the browser.

Need a clean restart

docker-compose down
docker-compose up -d --build
